# Тест-кейсы

## Модуль расчета депозита

### TC-01: Расчет для пятницы вечером
**Предусловие:** Ресторан "Хачапури и Вино", рейтинг 4.8
**Шаги:**
1. Выбрать дату — пятница
2. Выбрать время — 19:00
3. Нажать "Рассчитать"
**Ожидаемый результат:** Депозит = 30% (900₽ при среднем чеке 3000₽)

### TC-02: Расчет для понедельника днем
**Предусловие:** Тот же ресторан
**Шаги:**
1. Выбрать дату — понедельник
2. Выбрать время — 14:00
3. Нажать "Рассчитать"
**Ожидаемый результат:** Депозит = 10% (300₽)

## Модуль оплаты

### TC-03: Успешная оплата
**Шаги:**
1. Забронировать столик
2. Ввести корректные данные карты
3. Нажать "Оплатить"
**Ожидаемый результат:**
- Сообщение об успехе
- Email с подтверждением
- Бронь в статусе "Оплачено"

### TC-04: Ошибка оплаты
**Шаги:**
1. Забронировать столик
2. Ввести неверные данные карты
3. Нажать "Оплатить"
**Ожидаемый результат:**
- Сообщение об ошибке
- Бронь не создается
- Гость может повторить оплату

## Модуль отмены

### TC-05: Отмена за 24 часа
**Предусловие:** Бронь создана, до визита > 24 часов
**Шаги:**
1. Зайти в личный кабинет
2. Нажать "Отменить"
**Ожидаемый результат:**
- Сообщение о возврате 100%
- Деньги вернутся на карту

### TC-06: Отмена за 2 часа
**Предусловие:** Бронь создана, до визита 2 часа
**Шаги:**
1. Зайти в личный кабинет
2. Нажать "Отменить"
**Ожидаемый результат:**
- Сообщение о возврате 50%
- 50% суммы вернутся на карту
