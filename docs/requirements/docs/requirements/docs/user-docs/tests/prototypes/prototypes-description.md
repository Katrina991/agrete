# Прототипы интерфейсов модуля «Умное бронирование»

## Инструмент: Figma

Прототипы созданы в Figma для визуализации ключевых экранов и пользовательских сценариев.

## Список прототипов

### 1. Главная страница поиска
- **Назначение:** Поиск ресторанов по городу, дате и количеству гостей
- **Ключевые элементы:** Поле поиска, календарь, селектор количества гостей, список ресторанов
- [Смотреть прототип](ссылка_на_figma_если_есть)

### 2. Карточка ресторана
- **Назначение:** Детальная информация о ресторане и выбор времени
- **Ключевые элементы:** Название, адрес, рейтинг, фото, сетка доступного времени
- **Важно:** При выборе времени отображается сумма депозита с пояснением

### 3. Экран расчета депозита
- **Назначение:** Показ итоговой суммы и правил расчета
- **Ключевые элементы:** Выбранное время, сумма депозита, иконка "i" с пояснением (динамический расчет в зависимости от дня/времени)
- **Состояния:** Обычный режим, режим с пояснением (тултип/попап)

### 4. Форма оплаты
- **Назначение:** Ввод данных карты для оплаты депозита
- **Ключевые элементы:** Поля для номера карты, срока, CVV, кнопка «Оплатить», сумма к оплате
- **Состояния:** Обычное, загрузка (спиннер на кнопке), ошибка валидации (подсветка полей)

### 5. Экран успешной оплаты
- **Назначение:** Подтверждение создания брони
- **Ключевые элементы:** Иконка успеха, сообщение «Бронь подтверждена», детали брони, кнопка «Вернуться к поиску»

### 6. Экран ошибки оплаты
- **Назначение:** Сообщение о проблеме с оплатой
- **Ключевые элементы:** Иконка ошибки, сообщение об ошибке, кнопки «Повторить» и «Изменить способ оплаты»

### 7. Личный кабинет гостя
- **Назначение:** Просмотр и управление бронями
- **Ключевые элементы:** Список броней (активные/истекшие), статус оплаты, кнопка «Отменить бронь» для активных

### 8. Админ-панель ресторана
- **Назначение:** Управление бронями для администратора
- **Ключевые элементы:** Список броней на сегодня, фильтры, кнопки «Подтвердить», «Отклонить», «Отметить приход»

### 9. Настройки правил депозита (для владельца)
- **Назначение:** Настройка процентов депозита по дням недели
- **Ключевые элементы:** Таблица с днями недели, поля для ввода процентов, кнопка «Сохранить»

## Состояния, проработанные в прототипах

| Экран | Состояния |
|-------|-----------|
| Поиск | Загрузка, результаты найдены, ничего не найдено |
| Карточка ресторана | Доступное время, недоступное время (занято) |
| Форма оплаты | Обычное, загрузка, ошибка валидации, ошибка сети |
| Оплата | Успех, ошибка, ожидание (202 Accepted) |
