# Глоссарий терминов

| Термин | Определение |
|--------|-------------|
| No-show | Неявка гостя без предупреждения |
| Депозит | Предоплата, гарантирующая бронь |
| Умный депозит | Динамический расчет суммы в зависимости от условий |
| Агрегатор столиков | Сервис поиска и бронирования ресторанов |
| PMS | Property Management System — система управления рестораном/отелем |
| Стейкхолдер | Заинтересованное лицо (гость, администратор, владелец) |
| Use Case | Вариант использования системы |
| BPMN | Нотация моделирования бизнес-процессов |
| API | Интерфейс для взаимодействия программ |
| JSON | Формат обмена данными |
