# Руководство пользователя
## Модуль «Умное бронирование»

### Для гостей

#### Как забронировать столик
1. Выберите город и дату
2. Найдите подходящий ресторан
3. Выберите время
4. Система покажет сумму депозита
5. Нажмите «Забронировать»
6. Введите данные карты
7. Получите подтверждение на email

#### Почему меняется депозит?
Депозит рассчитывается автоматически:
- **Пятница, суббота** вечером — 30% (высокий спрос)
- **Будни днем** — 10% (низкий спрос)
- **Воскресенье** — 20% (средний спрос)

#### Как отменить бронь
1. Зайдите в личный кабинет
2. Найдите активную бронь
3. Нажмите «Отменить»
4. Система покажет сумму возврата
5. Подтвердите отмену

### Для администраторов ресторана

#### Управление бронями
1. Войдите в админ-панель
2. Откройте раздел «Бронирования»
3. Вы увидите список на сегодня
4. Можно подтвердить или отклонить бронь

#### Настройка депозита
1. Перейдите в «Настройки»
2. Выберите «Правила депозита»
3. Задайте процент для каждого дня
4. Сохраните изменения
